Scripting RemoteAccess

You can call InterMapper RemoteAccess from a command line, and control a significant number of functions. This can be useful for automating the updating of maps, or for various testing purposes.

For detailed information on the use of the command-line for scripting InterMapper RemoteAccess, see Scripting RemoteAccess in the Advanced Importing and Exporting section of the User Guide's Reference section.
