Probe Tools Section

Use the <tools> section of a command-line probe to embed Companion Script code directly into a command-line probe. This keeps the script code and the probe code together, ensuring that you are running the correct version of the script.

<tool:scriptname>
  [script code]
</tool:scriptname>

Replace scriptname with the file name you want to use for the script.

Replace [script code] with the code of the script.

Note: When you load the probe, the script is written into the InterMapper Settings/Tools directory. If a script with the specified name exists, it is overwritten. For this reason, you should not edit scripts directly in the subdirectories of the Tools directory, since they are overwritten when probes are reloaded.

For detailed information, see Companion Scripts.
